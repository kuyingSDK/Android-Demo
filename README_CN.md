# 酷盈广告SDK Android示例Demo

[English Documentation](README.md) | [中文文档](README_CN.md)

## 项目简介

这是一个基于酷盈广告SDK的Android演示项目，展示了如何集成和使用酷盈广告SDK的各种广告格式。本项目为开发者提供了完整的广告集成示例和最佳实践。

## 功能特性

### 支持的广告类型

- **激励视频广告** - 用户观看完整视频后获得奖励的广告形式
- **插屏广告** - 全屏展示的广告，适合在应用自然断点处展示
- **开屏广告** - 应用启动时立即展示的广告
- **横幅广告** - 灵活的横幅格式，可出现在应用顶部、中间或底部
- **原生广告** - 包括原生广告、垂直滑动视频和贴片广告，与原生应用代码最兼容

## 快速开始

### 环境要求

- Android Studio 4.0+
- JDK 8+
- Android SDK API 21+
