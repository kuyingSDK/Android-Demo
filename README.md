# Kuying SDK Android Demo

[中文文档 / Chinese Documentation](README_CN.md) | [English Documentation](README.md)

## Project Introduction

This is an Android demo project based on Kuying advertising SDK, demonstrating how to integrate and use various ad formats of Kuying advertising SDK. This project provides developers with complete ad integration examples and best practices.

## Features

### Supported Ad Types

- **Rewarded Video Ads** - Users earn rewards after watching complete videos
- **Interstitial Ads** - Full-screen ads suitable for display at natural app breakpoints
- **Splash Ads** - Ads displayed immediately when the app launches
- **Banner Ads** - Flexible banner formats that can appear at the top, middle, or bottom of your app
- **Native Ads** - Including native ads, vertical swipe videos, and pre-roll ads, most compatible with your native app code

## Quick Start

### Requirements

- Android Studio 4.0+
- JDK 8+
- Android SDK API 21+
